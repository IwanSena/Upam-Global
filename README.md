## üöÄ Tentang Upam Global

**Upam Global** adalah panel admin portabel berbasis browser yang dirancang untuk mengelola konten, halaman, media, dan pengaturan tanpa backend server.  
Nama **Upam** adalah singkatan dari **Universal Portable Admin Manager**.

Proyek ini merupakan evolusi dari Upam Pro, dengan arsitektur yang lebih fleksibel dan siap digunakan di berbagai platform hosting statis seperti GitHub Pages, GitLab Pages, Cloudflare Pages, Netlify, dan lainnya.

## üß© Arsitektur Inti

Upam Global bekerja dengan 3 pilar utama:

1. **The Writer** ‚Äì Ekstensi browser di HP user  
2. **The Storage** ‚Äì GitHub/GitLab sebagai hardisk cloud  
3. **The Reader** ‚Äì Template web yang merender konten di browser pengunjung

Semua proses dilakukan secara client-side, tanpa server backend.
